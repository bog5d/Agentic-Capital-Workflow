> 硅基唐僧取经录：大模型终于长出了手脚，用Tool Use铲除资本市场的信息摩擦。

去年开始，我花了大概大半年时间专门研究 RWA (真实世界资产代币化)。因为我是硬科技背景，对金融市场的底层逻辑一直有浓厚兴趣。

在这个过程中，我发现了一个根本性问题：**资本市场的信息摩擦太大了。**

## 什么是信息摩擦？

简单说，就是"你知道的和我不知道的"之间的鸿沟。这个鸿沟越大，交易成本越高，市场效率越低。

传统金融用各种中介来弥合这个鸿沟——投行、律所、审计、评级机构。但这些都是人，人的认知有边界，精力有限，而且很贵。

## Tool Use：大模型终于长出了手

2024 年之前的大模型，像一个知识渊博但困在图书馆里的学者。它什么都知道，但什么都做不了。

Tool Use 改变了这一切。

大模型可以调用外部工具了：搜索网页、查询数据库、调用 API、执行代码。它不再只是"回答问题"，而是"执行任务"。

## 唐僧 + 孙悟空 = AI Agent

如果把 AI 比作唐僧——知识丰富但手无缚鸡之力，那 Tool Use 就是给他配上了孙悟空。

孙悟空能：
- **腾云驾雾** → API 调用，瞬间获取千里之外的数据
- **七十二变** → 多工具组合，灵活应对不同场景  
- **火眼金睛** → 数据分析，看穿数据背后的真相

有了孙悟空的唐僧，才真正具备了"取经"的能力。

## RWA + AI Agent

回到 RWA 领域。RWA 的核心痛点是信息不对称——资产方、资金方、监管方之间的信息鸿沟。

AI Agent 可以：
1. 自动抓取资产底层数据（房产登记、企业工商、司法诉讼）
2. 实时监控链上链下数据一致性
3. 自动生成尽调报告和风险评估

**信息摩擦被大幅降低，交易成本随之下降。**

这就是我在做的事情——用 AI Agent 铲除资本市场的信息摩擦。
